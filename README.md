# 简易自动信息发送器（AutoMessageSender）

✨ 一个使用 Python + Tkinter 制作的图形化自动信息发送工具。适合用于重复性聊天、教学演示、表情雨特效等用途。

---

## 🛠 功能介绍

- 自定义发送内容
- 设置发送间隔（单位：秒）
- 一键启动/停止
- 支持中文、英文、符号等任意字符
- 打包为 `.exe` 文件，免Python环境运行

---

## 🔧 技术栈

- `Python 3.11`（或你用的版本）
- `Tkinter` 图形界面
- `pyautogui` 实现自动输入
- `pyinstaller` 打包成 EXE
